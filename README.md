# Right Distractions - Art Portfolio

A beautiful, Apple-inspired portfolio website showcasing fine art paintings featuring sceneries, locations, and portraits.

## Deploy to GitHub Pages

### Quick Setup (5 minutes)

1. **Create GitHub Repository**
   - Go to [github.com](https://github.com) and create a new repository
   - Name it: `rightdistractions.github.io` (for free domain)
   - Or name it anything else for custom domain later

2. **Upload Your Files**
   - Upload all files from this folder to your repository
   - Files needed: `index.html`, `styles.css`, `script.js`

3. **Enable GitHub Pages**
   - Go to repository Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` / `master`
   - Folder: `/ (root)`
   - Click Save

4. **Visit Your Site**
   - **Free Domain**: `https://rightdistractions.github.io`
   - **Custom Domain**: Add your domain in Settings → Pages after buying it

### Custom Domain Setup (Optional)

1. **Buy Domain**: Purchase `rightdistractions.com` (~$12/year)
2. **DNS Settings**: Point to GitHub Pages servers
3. **GitHub Pages**: Add domain in Settings → Pages

## Customization

### Update Content
- Edit `index.html` for text changes
- Update `script.js` gallery data with your images
- Modify `styles.css` for color changes

### Add Your Images
1. Create an `images/` folder
2. Add your painting files
3. Update image URLs in `script.js`

### Contact Information
- Update email/phone in `index.html` contact section
- Modify form submission handler if needed

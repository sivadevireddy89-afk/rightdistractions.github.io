📋 How to Use Your Portfolio

  • Main Website: Open index.html for the full-featured portfolio
  • Add Your Images: Create an images folder and add your painting files
  • Update Content: Edit script.js to update gallery data with your actual paintings
  • Customize Colors: Modify CSS variables in styles.css to match your brand
  • Contact Form: Update contact information in the HTML file
  • Full Features: The main site includes filtering, modal views, smooth scrolling, and responsive design
